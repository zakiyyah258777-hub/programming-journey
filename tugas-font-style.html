Praktik CSS Font Properties

5 Paragraf dengan Kombinasi Font Berbeda

Paragraf 1: Font size 20px, font weight normal (400), font style normal, dan font family Arial. Contoh penggunaan font dengan satuan pixel yang fixed size.

Paragraf 2: Font size 1.5em, font weight bold (keyword), font style italic, dan font family Times New Roman. Satuan em bersifat relatif terhadap font size parent element.

Paragraf 3: Font size 1.8rem, font weight 700 (bold), font style oblique, dan font family Courier New. Satuan rem bersifat relatif terhadap root element (html).

Paragraf 4: Font size 130%, font weight 300 (light), font style normal, dan font family Georgia. Satuan persen (%) bersifat relatif terhadap font size parent element.

Paragraf 5: Font size large (keyword), font weight 900 (black/bolder), font style italic, dan font family Verdana. Menggunakan keyword untuk font size yang sudah ditentukan oleh browser.

Daftar Kelas CSS yang Tersedia

Font Size Classes:

.px-size - Font size 18px

.em-size - Font size 1.5em

.rem-size - Font size 1.8rem

.persen-size - Font size 120%

.keyword-size - Font size large

Font Weight Classes:

.weight-light - Font weight 300

.weight-normal - Font weight 400 (normal)

.weight-bold - Font weight 700 (bold)

.weight-bolder - Font weight 900

.weight-keyword - Font weight bold (keyword)

Font Style Classes:

.style-normal - Font style normal

.style-italic - Font style italic

.style-oblique - Font style oblique

Font Family Classes:

.font-arial - Arial, Helvetica, sans-serif

.font-times - Times New Roman, Times, serif

.font-courier - Courier New, Courier, monospace

.font-georgia - Georgia, Times New Roman, serif

.font-verdana - Verdana, Geneva, Tahoma, sans-serif
