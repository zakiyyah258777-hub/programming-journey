TENTANG EKSTERNAL CSS

Eksternal CSS adalah CSS yang ditulis di file terpisah (.css) dan dihubungkan ke HTML untuk mengatur tampilan halaman. Penjelasan singkat: CSS disimpan di luar file HTML Digunakan oleh banyak halaman sekaligus Membuat kode lebih rapi, terstruktur, dan mudah dirawat
